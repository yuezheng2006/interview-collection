## 项目结构说明

每个面试者的项目都包含以下标准结构：
- `frontend/` - 前端项目（Vue 3 + TypeScript）
- `backend/` - 后端服务（Go）
- `doc/` - 项目文档
- `README.md` - 项目说明
- `env.example` - 环境变量示例

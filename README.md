## 项目结构说明

每个面试者的项目都包含以下结构：

**全栈Web应用项目**（如 chenhuiyuan）：
- `frontend/` - 前端项目（Vue 3 + TypeScript）
- `backend/` - 后端服务（Go）
- `doc/` - 项目文档
- `README.md` - 项目说明
- `env.example` - 环境变量示例

**Chrome扩展项目**（如 lijiarui）：
- `ai-code-review-to-github/` - Chrome扩展插件文件
- `img/` - 效果截图
- `使用说明.md` - 使用指南
- `AI代码审查插件评分报告.md` - 评分报告
